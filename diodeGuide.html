⚡ ダイオード早見表 / リファレンス

ダイオードとは、電流を一方向にだけ流す半導体素子。 記号の三角形が指す向き（アノード A → カソード K）が順方向。 用途・性能の違いで多くの種類があり、選び方を間違えると「発熱する」「動かない」「壊れる」原因になります。 ※ 本ページの電圧・電流値は一般的な目安です。実機ではデータシートを確認してください。

🧮 LED 抵抗値かんたん計算

電源電圧・LEDの順方向電圧(Vf)・流したい電流を入れると、必要な抵抗値を計算します。
※ 電源電圧は Vf より 0.5〜1V 以上高めが目安。例: 白(3.6V)・青(2.8V)を 3.3V で点けるのは厳しい（届かず点灯しない/暗い）。赤・黄・橙なら 3.3V でもOK。
※ 流したい電流の目安: （暗い）5mA 〜 20mA（明るい）。明るさは電流で変わります。細かく調整する場合は PWM（高速点滅で平均を変える）などを使います。

記号 ▷| の縦棒側もカソード。

🗂 種類ごとの詳細

一般整流ダイオード Rectifier

Vf ≈ 0.7〜1.0V / 50〜1000V耐圧
主な用途
AC→DCの整流、電源の逆接続（逆挿し）保護、フリーホイールダイオード。
入手しやすい例
1N4007（1A/1000V・最も万能）、1N5408（3A）。
他との違い
安くて耐圧が高い反面、逆回復が遅く高周波スイッチングには使えない。商用50/60Hz整流の定番。

ショットキーバリア Schottky / SBD

Vf ≈ 0.2〜0.45V / 高速・低損失
主な用途
DC-DCコンバータの整流、逆流防止（電源OR）、電圧降下を抑えたい箇所。
入手しやすい例
1N5819（1A/40V・リード品）、1N5819W（同スペックのSMD版/SOD-123）、SS34（SMD・3A/40V）、1N5822（3A）。
他との違い
Vfが低く発熱が少ない・高速。ただし逆漏れ電流が大きく耐圧が低め、高温で漏れが増える点に注意。

小信号スイッチング Small-signal

Vf ≈ 0.7V / 〜200mA / 数ns
主な用途
信号のクランプ、ロジック保護、小電流整流、検波。電子工作の万能品。
入手しやすい例
1N4148（定番）、1N914（ほぼ同等）。
他との違い
小型・超高速だが電流容量が小さい。大電流の整流には使えない。迷ったらまずこれ。

ファストリカバリ整流 Fast Recovery

Vf ≈ 0.9〜1.3V / 逆回復が速い
主な用途
スイッチング電源、インバータなど高周波回路の整流。
入手しやすい例
UF4007（1A/1000V）、FR107、MUR120。
他との違い
一般整流（1N400x）より逆回復が速い。ショットキーより耐圧を取りやすいがVfは高め。

ツェナーダイオード Zener

逆方向の降伏電圧を利用（3.3〜100V）
主な用途
定電圧・基準電圧の生成、過電圧クランプ、簡易レギュレータ。
入手しやすい例
1N4733A（5.1V）、1N4742A（12V）、BZX55C シリーズ。
他との違い
普通のダイオードと逆向き（カソードを＋側）に使い、降伏電圧で電圧を固定する。直列に電流制限抵抗が必須。

TVS（サージ保護） TVS / Transient

クランプ電圧で規定 / 大電力サージ吸収
主な用途
ESD・落雷サージ・過電圧からICや入力ポートを守る。USB/電源ライン保護。
入手しやすい例
P6KE 系（0.6kW）、1.5KE 系（1.5kW）、SMAJ 系（SMD）。
他との違い
ツェナーに似るが、瞬間的な大電力を吸収する保護専用設計で応答が極めて速い。双方向品もある。

LED（発光ダイオード） LED

Vf: 赤≈1.8V / 青・白≈3.0〜3.4V
主な用途
状態表示、イルミネーション、照明、光通信の送信側。
入手しやすい例
3mm/5mm 砲弾型（赤・緑・青・白・黄）、各種SMD（0603/2012 等）。
他との違い
順方向で発光する。必ず電流制限抵抗か定電流回路が必要。逆耐圧が低く逆接続に弱い。

ブリッジダイオード Bridge Rectifier

整流ダイオード4本内蔵 / 全波整流
主な用途
ACを全波整流してDC化。電源回路のフロントエンド。
入手しやすい例
DB107（1A）、KBP210（2A）、KBPC 系（大電流）。
他との違い
4本を1パッケージ化。AC×2・DC(+/−)×2端子で配線が簡単。極性表示（〜と+−）を確認。

定電流ダイオード CRD

動作範囲内で電流ほぼ一定
主な用途
LEDを電圧変動に強く一定の明るさで点灯、簡易な定電流源。
入手しやすい例
E-103（約10mA）、E-153（約15mA）。
他との違い
整流素子ではなくJFETを使った定電流素子。抵抗1本のLED駆動より明るさが安定する。

バラクタ（可変容量） Varactor

逆電圧で接合容量が変化
主な用途
ラジオ等の同調、PLL/VCOの周波数制御。
入手しやすい例
1SV149、1SV101 等。
他との違い
整流ではなく「電圧で容量が変わるコンデンサ」として使う特殊用途。

フォトダイオード Photodiode

光が当たると逆方向に光電流
主な用途
光検出、明るさセンサ、光通信の受信側、リモコン受光。
入手しやすい例
各種フォトダイオード／フォトトランジスタ、受光モジュール。
他との違い
LEDの逆動作にあたる受光素子。光量に応じた電流を取り出す。

🧭 用途から選ぶ（逆引き）

電源を整流したい（AC→DC）
→ 一般整流（1N4007）／配線を楽にするならブリッジ（DB107）。
効率よく整流したい・電圧降下を抑えたい
→ ショットキー（1N5819 / SS34）。DC-DCの整流に最適。
電源の逆接続（逆挿し）を防ぎたい
→ 直列にショットキー、またはPchMOSFETによる理想ダイオード。
高速な信号を扱う・小電流整流
→ 小信号スイッチング（1N4148）。
スイッチング電源の高周波整流
→ ファストリカバリ（UF4007）またはショットキー。
一定の電圧を作りたい・基準電圧
→ ツェナー（例: 5.1Vなら 1N4733A）＋電流制限抵抗。
サージ・ESDから守りたい
→ TVS（P6KE / SMAJ 系）を保護したいラインに並列。
LEDを安定した明るさで点けたい
→ 電流制限抵抗、または定電流ダイオード（E-153）。

📡 ESP32 でよく使うダイオード

ESP32（および S2/S3/C3 系）は 3.3Vロジック。GPIOは5Vトレラントではない（＝5Vを直接つなぐと壊れる。下の用語解説参照）ので、保護とレベルに特に注意。代表的な使いどころを整理します。

電源OR / 逆接続保護 USB + 外部電源

ショットキーで低損失に
用途
USB(5V)と外部電源/電池をダイオードORでつなぎ、どちらが挿さっても給電。VINの逆挿し保護にも。
おすすめ型番
SS14 / 1N5819（1A級）、MBR0520（SMD・0.5A）、BAT54C（小電流OR）。
ポイント
Vfが低いショットキー必須。電流が大きい用途は理想ダイオード(Pch MOSFET)も検討。多くのDevKitはUSB5VとVBUSの間にこの構成。

USB データラインのESD保護 D+/D−

TVSダイオードアレイ
用途
ESP32-S2/S3/C3 のネイティブUSBや、CH340/CP2102のD+/D−をESD/サージから保護。
おすすめ型番
USBLC6-2SC6（USB用ESD保護の定番）、PESD5V0 系。
ポイント
低容量タイプを選ぶと信号品質を損なわない。市販ボードもほぼこの石を採用。

リレー/モーター駆動のフライホイール Flyback

逆起電力（コイルが出す逆向きの電圧）を吸収
用途
ESP32のGPIO→トランジスタ/MOSFET経由でリレーやソレノイドを駆動する際、コイルに並列で逆起電力を逃がす。
おすすめ型番
1N4007（リレー定番）、小型なら 1N4148。高速復帰なら UF4007。
ポイント
コイルに対しカソードを＋側へ。これが無いとスイッチング素子やESP32が破損しやすい。リレーモジュールは内蔵済みが多い。

GPIO入力の過電圧クランプ 3.3V保護

5V信号 → 3.3Vへ保護
用途
外部の5V信号やノイズからGPIOを守る。ショットキーで3.3Vレールへクランプ、または直列抵抗＋クランプ。
おすすめ型番
BAT54 / BAT54S（クランプ）、入力保護なら PESD3V3 系TVS。
ポイント
確実な5V→3.3V変換はレベルシフタ(例: TXS0108E)を推奨。ダイオードクランプは簡易用途向け。

RTC / バックアップ電源のOR Deep-Sleep / 電池

低Vf・低漏れ
用途
主電源とコイン電池/スーパーキャパシタをORし、停電時もRTCやSRAMを保持。
おすすめ型番
BAT54C（2素子OR）、低漏れショットキー。
ポイント
逆漏れ電流が小さい品を選ぶと電池消費を抑えられる。Vf分の電圧降下に注意。

ステータス表示LED Indicator

GPIOは1ピン最大〜数十mA
用途
通信状態や電源の表示。GPIOで直接、または明るさ重視ならトランジスタ駆動。
おすすめ
各色LED＋電流制限抵抗（3.3Vなら赤で約150〜330Ω目安）。WS2812(NeoPixel)もLEDの一種。
ポイント
ESP32のGPIO総電流には上限あり。多数点灯やパワーLEDは外部ドライブで。

📖 用語解説（むずかしい言葉）

アノード / カソード
ダイオードの2本の足の名前。電流が入る＋側がアノード(A)、出る−側がカソード(K)。本体の帯がある側がカソード。
順方向 / 逆方向
電流が流れる向き（A→K）が順方向。その逆が逆方向で、ふつうは電流を止める（＝一方通行）。
定格電流（1A・3A など）
順方向に流し続けられる電流の上限（単位アンペア）。超えると発熱して壊れる。実際に流す電流より余裕のある品を選ぶ。
耐圧（1000V など）
逆方向にかけられる電圧の上限（単位ボルト）。これを超えると降伏して壊れる。電流とは別の指標。
順方向電圧 Vf
順方向に電流を流したとき、ダイオードで失われる電圧。低いほど発熱が少なく効率的（ショットキーが低い）。
逆回復（時間）
順方向→逆方向に切り替わるとき、止まりきるまでの遅れ。短いほど高速。一般整流は遅く、ショットキー/高速品は速い。
降伏（ブレークダウン）
逆方向に強い電圧をかけると、ある電圧で急に電流が流れ出す現象。ツェナーはこれをあえて使って電圧を一定にする。
クランプ
電圧が一定値を超えないよう「頭打ち」にすること。過電圧から回路を守る使い方。
ダイオードOR
複数の電源をダイオードでつなぎ、電圧の高い方から自動で給電する回路。逆流は各ダイオードが防ぐ。
理想ダイオード
MOSFETと制御ICで作る、Vf（電圧降下）がほぼ0のダイオード相当回路。大電流でも発熱が少ない。
逆起電力
コイル（リレー・モーター）への電流を切った瞬間に出る、逆向きの高い電圧。放置するとスイッチ素子を壊す。
フライホイール（フリーホイール）ダイオード
コイルに並列に入れ、上の逆起電力を逃がして消すためのダイオード。「還流ダイオード」とも言う。
5Vトレラント
3.3Vで動くピンに5Vをつないでも壊れない耐性のこと。ESP32のGPIOはこれが無いので5V直結はNG。
レベルシフタ
3.3Vと5Vなど、電圧の違う信号同士を安全につなぐ変換IC。確実な電圧変換にはこちらを使う。
ESD（静電気放電）
人体などにたまった静電気が一瞬で流れる現象。数千Vになり半導体を壊す。保護にTVSを使う。
サージ
落雷や電源の切替などで瞬間的に発生する、異常に高い電圧/電流のこと。
TVS
ESDやサージを一瞬で吸収する保護専用ダイオード（Transient Voltage Suppressor）。
逆接続保護（逆挿し保護）
電源の＋−を逆に挿しても回路が壊れないようにする仕組み。ショットキーやMOSFETで作る。
電流制限抵抗
LEDなどに流れる電流を抑えるために直列に入れる抵抗。無いと過電流で焼ける。
デッドゾーン
スティックの中央付近など、わずかな入力を「無し」と扱う不感帯。誤動作防止に使う。
値は一般的な目安です。実際の設計ではメーカーのデータシート（最大定格・Vf・逆回復時間・許容損失）を必ず確認してください。
