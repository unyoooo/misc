🎮 ゲームパッドが認識されない場合のチェックリスト（クリックで展開）
  1. このページにフォーカスを当てたまま、ゲームパッドのボタンをどれか1回押す（Chromeの仕様で、ボタン押下があるまで認識されません）
  2. USBケーブルで接続しなおすか、Bluetooth再接続すれば治ることが多い
  3. Windowsのコントロールパネル → デバイスマネージャー →「ヒューマンインターフェイスデバイス」にHID準拠ゲームコントローラーが出ているか確認
  4. 出ていない場合、DualSenseが「オーディオデバイス」や「HIDキーボード」として誤認識されている可能性。いったんWindowsから削除してペアリングし直す
⚠ このページは iframe 内で動作しています。
iframe 内ではブラウザのセキュリティ仕様により Gamepad API が使えません。
ゲームパッドを検出するには、下記のパスをエクスプローラーに貼り付けて、Chrome等のブラウザで直接ファイルを開いてください：
C:\Users\unyoo\cloude_code\KeyInputTest\index.html
直接開いた後、ゲームパッドのいずれかのボタンを1回押すと認識されます。
